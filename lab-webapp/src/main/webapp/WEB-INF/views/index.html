雨无声实验室 创意与创造的地方

雨无声实验室是广西大学雨无声网站推出的一个展现新技术，新产品的平台。 目的在于开发出更多服务于或者方便于西大学生的产品，也给热爱技术的同学们提供一个展现自己的平台。 我们希望在这个实验室里毕业出很多优秀的产品，提供优质的服务。

雨无声信息发布平台 轻松获取校园一手信息

雨无声信息发布平台是一个服务于广西大学师生的网站。任何校内的组织，学院，比如协会经过认证后都可以在这上面发布信息。任何人都可以通过信息发布平台和Android客户端获取到信息:)

雨文分享平台 西大文库，你值得拥有！

雨文分享平台是一个由西大人共建，共享的一个专门服务于广西大学师生的文档分享平台。同学们可以把自己日常生活，学习，与考试中相关的资料放在这里与大家共享，共同进步，共同打造我们具有西大特色的西大文库！

欢迎来到 雨无声实验室 开源项目主页

雨无声实验室是一个面向于西大师生的服务型平台。提供一系列实际生活，学习等方面的便捷服务。另一方面，实验室也乐意提供给那些渴望展现自己的能力，分享自己的创意的同学。在实验室，不管是你是雨无声人，还是西大人，都可以参与进来.不要担心自己是电脑盲，计算机技术只是实现创意的一种手段，我们需要的是创意。把你的想法，能力共享出来，雨无声人，西大人，会技术的，不会技术的，我们一起来孵化。

如果你已经安装了git的话，那么打开你的终端，输入下面的命令即可得到一个完整的版本库

$ cd your_repo_root/repo_name
$ git clone https://github.com/longkai/lab.newgxu.cn.git  #克隆远程版本库至本地
$ git fetch  #抓取远程版本库的最新更新

你也可以简单的点击上面的压缩包下载项目的最新源码，但是那样会失去一些版本控制历史信息。

设计理念

基于伪插件式的体系吧:)，封装了一个核心类库，然后所有的项目都调用该库提供的核心功能。并且作为一个扩展组件插入到原有的项目中。具体见代码，希望大伙能一起讨论并实现真正意义上的插件体系:)

开发技术

服务端采用流行的java web mvc开发。基于springmvc，jpa以及一些流行的开源项目组成。前端使用了bootstrap+jquery展现。采用maven管理项目。

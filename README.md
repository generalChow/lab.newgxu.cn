## 雨无声实验室
雨无声实验室是广西大学雨无声网站推出的一个展现新技术，新产品的平台。
目的在于开发出更多服务于或者方便于西大学生的产品，也给热爱技术的同学们提供一个展现自己的平台。
我希望在这个实验室里毕业出很多优秀的产品。

## 开发技术
Java(springmvc, jpa等)

## 设计理念
采用了插件式的设计理念，任何一个项目在实验室上都是以即插即用的形式使用。
每一个插件都有自己独立的命名空间（config，entity，repository，service，controller等）
有点“伪”，但是确实实现了:)
